# 概要

browsing-memory のブラウザ拡張機能の使い方

インストールと初期設定が済んでいることが前提です。まだの場合は [installation.md](./installation.md) を参照してください。

## 基本的な使い方

基本的にはブラウザでいつも通りウェブを巡回するだけです。自動保存が有効になっていれば、閲覧したページが自動的に Markdown ノートとして保存先ディレクトリに記録されていきます。

ファイルは `2026-06-27_14-30-00.md` のように「保存した日時」をファイル名として保存され、ページのタイトル・URL などの情報（フロントマター）と本文、AI 要約（有効な場合）が記録されます。

## 保存されるタイミング

ページはアクセスした瞬間ではなく、**そのページから離れたとき** に保存されます。具体的には次のいずれかのタイミングです。

- 別のページに移動したとき
- 別のタブに切り替えてそのページから離れたとき
- そのページのタブを閉じたとき

> **注意**
> ページを開いてからすぐに閉じたり、読み込みが終わる前に離れたりした場合は、保存されないことがあります。

> **保存先が未設定の場合**
> 保存先ディレクトリが未設定のまま自動保存が走ると、ツールバーのアイコンに「!」マークが表示され、保存先を促す設定画面が自動的に開きます。このときそのページは保存されず、後から保存先を設定しても遡って保存されることはありません。記録を始める前に、[installation.md](./installation.md) の「デフォルト保存ディレクトリの設定」を参照して保存先を指定しておいてください。

## 記録（自動保存）をOFFにする

しばらく記録を取りたくないときは、自動保存をOFFにできます。

1. ブラウザ右上のツールバーにある **Browsing Memory のアイコン** をクリックします。
2. 表示されるポップアップの **「自動保存」** トグルをOFFにします。

OFFにしている間はページの記録が一切行われません。再び記録したくなったら、同じトグルをONに戻してください。

## 特定のサイトだけ記録対象外にする（ドメインフィルタ）

自動保存はONのままにしつつ、特定のサイトだけ記録したくない場合はドメインフィルタを使います。

1. 拡張機能のオプション（設定）画面を開きます。
2. 「一般設定」の **「ドメインフィルタ」** の入力欄に、除外したいドメイン（例: `example.com`）を入力し、**「追加」** をクリックします。
3. 追加したドメイン（およびそのサブドメイン）のページは自動保存の対象外になります。

除外対象のサイトを開いている間は、ツールバーのアイコンがグレーアウト表示になり、自動保存されないことが分かります。

登録したドメインを記録対象に戻したいときは、ドメインフィルタの一覧から該当ドメインの **削除** ボタンをクリックしてください。

## （オプション）外部のLLMで要約する

要約は標準で Chrome 組み込みの Summarizer API を使うため、特別な設定なしで利用できます。要約の品質をより重視したい場合は、外部のLLM（OpenAI など）を使うこともできます。**この設定は必須ではありません。**

設定は拡張機能のオプション画面の「AI要約」→ **「カスタムモデル」** セクションから行います。大まかな流れは「プロバイダーを追加する」→「モデルを追加する」の2ステップです。

### プロバイダーの追加

1. **「AI要約を有効化」** のトグルスイッチをオンにします。
2. 「プロバイダー」項目の **「＋ プロバイダーを追加」** をクリックします。
3. ポップアップで使いたいプロバイダーを選び、必要な項目（APIキーなど）を入力します。プルダウンからプリセットのプロバイダーを選ぶと、プロバイダー名や Base URL は自動的に入力されます。
4. **「保存」** をクリックします。

### モデルの追加

1. 「モデル」項目の **「＋ モデルを追加」** をクリックします。
2. ポップアップで、先ほど追加したプロバイダーと、使いたいモデルを選びます。「人気のモデル」の一覧から選ぶと、モデル名とモデルIDは自動的に入力されます。
3. **「保存」** をクリックします。

以降の自動保存では、ここで設定した外部LLMによる要約が使われます。外部LLMでの要約に失敗した場合などは、Chrome 組み込みの要約がフォールバックとして使用されます。

> **例：OpenAI を使う場合**
> - プロバイダー追加時に、プルダウンから **「OpenAI」** を選択し、**APIキー** にご自身の OpenAI の APIキーを入力します。
> - モデル追加時に、プロバイダーとして **「OpenAI」** を選び、「人気のモデル」から **GPT-4o** などを選択します。
>
> 必要な項目（APIキーの要否など）はプロバイダーによって異なります。上記はあくまで OpenAI を使う場合の例です。
