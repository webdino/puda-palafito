# 概要

browsing-memoryのブラウザ拡張機能インストール方法

## インストール方法

### 1. `browsing-memory-1.0.0-chrome.zip` の解凍

[browsing-memory-1.0.0-chrome.zip](../browsing-memory-1.0.0-chrome.zip)の解凍を行ってください。

### 2. Chromeの拡張機能管理画面を開く

1. Google Chromeを起動します。
2. アドレスバーに `chrome://extensions/` と入力し、Enterキーを押して拡張機能の管理画面を開きます。
3. 画面右上の **「デベロッパー モード」** のトグルスイッチをオンにします。

### 3. 拡張機能をパッケージ化されずに読み込む

1. 画面左上の **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
2. ファイル選択ダイアログが表示されるので、事前に解凍しておいた `browsing-memory-1.0.0-chrome` フォルダを選択し、**「フォルダーの選択」** をクリックします。
3. 拡張機能の一覧に「Browsing Memory」が追加され、有効になっていることを確認します。

### 4. 拡張機能のオプション画面から各種設定を行う

1. 追加された「Browsing Memory」カード内にある **「詳細」** ボタンをクリックします。
2. 詳細ページ内の **「拡張機能のオプション」** をクリックして設定画面を開きます。

#### 4-1. デフォルト保存ディレクトリの設定

1. 設定画面の「一般設定」にある「デフォルト保存ディレクトリ」の **「参照」** ボタンをクリックします。
2. フォルダ選択ダイアログで、保存先に指定したいフォルダを選択し、**「フォルダーの選択」** をクリックします。
3. ブラウザ上部に表示される「このサイトへのファイルの編集を許可しますか？」という確認ポップアップで **「許可する」** を選択します。

#### 4-2. AI要約の設定（ローカルモデルのダウンロード）

1. 設定画面の左メニューから **「AI要約」** を選択します。
2. 「Chrome組み込みの要約モデル」のセクションにある **「モデルのダウンロード・動作検証」** ボタンをクリックします。
3. モデルのダウンロードが開始されるので、プログレスバーが **100%** になり、「動作検証が完了しました。要約機能が利用できます。」と表示されるまで待ちます。
