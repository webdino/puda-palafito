# Claude デスクトップアプリ：MCPサーバー（拡張機能）追加手順書

## 1. 設定画面へのアクセス

1. Claudeデスクトップアプリの画面左下にある**ユーザー名（アカウント名）**をクリックします。
2. メニューから「**設定**」を選択します。
3. 設定画面の左メニューから「**拡張機能**」を選択し、画面右上にある「**拡張機能を参照**」ボタンをクリックします。

## 2. 拡張機能ファイルの選択とインストール

1. 一覧から「**Browsing Memory**」を選択します。
2. 「拡張機能をインストール」ウィンドウが開いたら、「**ローカルの未検証拡張機能をインストール**」ボタンをクリックします。
3. ファイル選択ダイアログが開くので、browsing-memory.mcpbを選択して「**開く**」をクリックします。
4. セキュリティの確認ポップアップ（「Browsing Memoryをインストールしますか？」）が表示されたら、「**インストール**」を選択します。

## 3. 保存先ディレクトリ（WebClip-Directory）の設定

1. インストール完了後、詳細設定のポップアップが表示されます。
2. `WebClip-Directory` の項目にある「**参照**」ボタンをクリックします。
3. エクスプローラー（またはFinder）が開くので、データを保存・管理するための任意のローカルフォルダ
4. 設定ウィンドウの右下にある「**保存**」ボタンをクリックして確定します。

## 4. プロジェクトの新規作成

1. ホーム画面の左メニューから「**プロジェクト**」を選択します。
2. 右上にある「**新規プロジェクト**」ボタンをクリックします。
3. プロジェクト名に `browsing-memory` と入力し、「**プロジェクトを作成**」をクリックします。

## 5. プロジェクト指示（カスタム指示）の設定

1. 作成したプロジェクト画面内にある「**指示を設定**」をクリックします。
2. 「プロジェクトの指示を設定」の入力欄に、[Prompt.md](./prompt.md)を記述します。
3. 記述後、右下の「**変更を保存**」をクリックします。
