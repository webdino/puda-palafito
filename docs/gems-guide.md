# Gems セットアップガイド

このドキュメントでは、**puda-palafito** で Google Drive に保存したデータを Gemini の Gems に読み込ませるためのセットアップ手順を説明します。

## 前提条件

Gems の設定を始める前に、以下が完了していることを確認してください。

- 拡張機能が Chrome に読み込まれている
- Google Drive API の設定が完了している
- 拡張機能の設定画面で Google Drive 連携が完了している
- 保存先フォルダが設定され、記録データが Google Drive に保存されている

Google Drive の設定がまだの場合は、先に [Google Drive API の設定手順](./google-drive-api-setup.md) と [ユーザーガイド](./user-guide.md) を参照してください。

## Gems の設定手順

### 1. Gem 管理画面を開く

Gemini の左サイドバーにあるメニューから操作します。

1. 画面左上の **「作成したもの」** セクションを探します。
2. **「Gem >」** と書かれたメニュー項目をクリックします。
3. 画面中央または右側に Gem の管理画面が開きます。

### 2. 新しい Gem を作成する

1. **「マイ Gem」** セクション右側の **「+ Gem を作成」** をクリックします。

### 3. カスタム指示を設定する

Gem の作成画面で使用するカスタム指示は、拡張機能の設定画面からコピーしてください。

1. 拡張機能の設定画面を開きます。
2. **「Gemsの設定」** セクションにあるカスタム指示欄を表示します。
3. **コピー** ボタンを押して、表示されているテキストをコピーします。
4. Gemini の Gem 作成画面に戻り、カスタム指示欄へ貼り付けます。

### 4. Google Drive 上のファイルを知識として追加する

1. **「Gem が参照するファイルを追加します」** の横にある **「+」** アイコンをクリックします。
2. **「Google ドライブ」** を選択します。
3. 拡張機能が自動作成したフォルダを開きます。
4. フォルダ内の記録ファイルを選択します。

![Gems設定画面](./images/gems-settings.png)

## 5. 保存する

1. 画面右上の **「保存」** をクリックして完了です。

## 6. 使い方イメージ

Gems の設定が完了すると、Google Drive に保存された記録ファイルを知識として参照しながら会話できるようになります。

この動画では、保存済みのニュース記事を知識として追加した Gem に質問している例を紹介します。

質問例:

- 「最近見ているニュースの傾向を教えて」
- 「Z世代のニュースの内容もう少し具体的に教えて」
- 「そのURLを教えて」
